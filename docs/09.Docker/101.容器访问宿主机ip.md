在 Docker 20.10 及以上版本中，Docker 访问宿主机的方式有所变化：

在 Docker 命令行中，添加：

```bash
--add-host=host.docker.internal:host-gateway
```

在 docker-compose 中，添加：

```bash
extra_hosts:
- host.docker.internal:host-gateway
```

即可通过 host.docker.internal 域名访问宿主机。
